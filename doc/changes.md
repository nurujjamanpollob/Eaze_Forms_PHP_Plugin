# Changelog

All notable changes to this project will be documented in this file.

## Update - 4 January 2026

### Website
- **Admin Panel**: Implemented a fully responsive design using Tailwind CSS, optimized for all device sizes.

### Settings
- **Mail Templates**: Added customization for HTML mail templates for both user form submission confirmations and admin notifications.
- **Pagination**: Added configurable pagination sizes for form submissions on the dashboard and management pages.

### Database
- **Default Values**: Introduced new default values for pagination settings.
- **Schema Enhancements**: Added support for HTML Mail template customization fields for User and Admin notifications.

### General
- **Security**: Successfully completed a comprehensive security audit; no threats or vulnerabilities identified.

---

## Initial Release - 1 January 2026

- **Launch**: Initial version of the Eaze Contact Form System.
- **Features**: Core submission handling, SQLite integration, and basic admin interface.
